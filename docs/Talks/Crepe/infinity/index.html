The Vanishing Trick

Consider the set of natural numbers, starting with 0:

\(\{0,\, 1,\, 2,\, 3,\, 4,\, \dots\}\)

What happens when we add 1 to every number? We get \(\{1,\, 2,\, 3,\, \dots\}\). The same set — but we've lost 0!

In fact, adding \(n\) will remove \(n\) numbers!

This is only possible because we have an infinite list. With finitely many objects in finitely many boxes, if you can perfectly fill the boxes once, you can never rearrange and end up with an empty box.

The Doubling Trick

Start with the natural numbers \(\N = \{0,\, 1,\, 2,\, 3,\, \dots\}\). Split them into evens and odds.

Even numbers Odd numbers

Each half is actually identical to the original set, just relabeled!

Divide the evens by 2 → you get \(\N\) again. Subtract 1 from the odds, then divide by 2 → a second complete copy of \(\N\)!

One infinity just became two infinities. This doubling phenomenon is at the heart of the Banach–Tarski Paradox.
