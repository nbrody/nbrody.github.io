Symmetries

A Symmetry of a Space

Now that we've seen some spaces, let's generalise our "integer-translations on the line" example.

A symmetry (or isometry) of a space is any rigid motion — a rearrangement that preserves all distances.

In the plane, these include translations, rotations, and reflections.

Inverses & Identity

Notice: every rigid motion can be undone by another rigid motion.

We should also consider the identity map — the "do nothing" transformation that leaves every point in place — as a (trivial) symmetry.

Click Apply to see the isometry, then Undo to see its inverse.

Composing Symmetries

Given any symmetry, we can iterate it and get a new one!

More generally, given two symmetries we can compose them: apply one, then the other.

The Placemat

Consider a rectangular placemat. We can rotate it 180°, or flip it horizontally or vertically.

Notice: the order in which we apply symmetries affects the outcome!

But when we compose symmetries, we always get another symmetry.

Groups from Generators

The group generated by some symmetries is the collection of all symmetries we can build by composing our starting ones in every possible way.

These starting symmetries are building blocks — we combine them in many, many ways to build a whole group of symmetries.

When Order Doesn't Matter

Sometimes the order we compose symmetries doesn't matter!

For example, two translations of the plane always commute: it doesn't matter which one we do first.
