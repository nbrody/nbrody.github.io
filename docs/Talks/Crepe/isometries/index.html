Symmetries

The Euclidean Plane

Here is ℝ², the Euclidean plane — the most familiar geometric space.

We have two basis vectors: e₁ pointing to the right and e₂ pointing upward.

Every point can be described by its coordinates (x, y).

This is our stage. Next, we'll see what happens when we start moving every point at once.

Translations

A translation slides every point of the plane by the same amount in the same direction.

We start with two basic ones:

T₁ — translate one unit to the right
T₂ — translate one unit up

Each translation is an isometry — it preserves all distances. Nothing stretches or rotates.

Combining Translations

We can not only iterate a translation (apply it again and again), but we can also combine two separate translations!

Applying T₁ a total of m times and T₂ a total of n times gives a new translation to the point (m, n).

The combined translation is just one big slide — the composition of the individual steps. The order doesn't matter: m right + n up = n up + m right!

The Placemat

Consider a rectangular placemat. We can rotate it 180°, or flip it horizontally or vertically.

Notice: the order in which we apply symmetries affects the outcome!

But when we compose symmetries, we always get another symmetry.

Sometimes the order we compose symmetries doesn't matter!
