Space

What does it mean to measure distance?

Measuring Distance

Mathematics begins with abstraction. We take a familiar idea — distance — and strip it to its absolute essentials.

A set of points where a distance \(d(x,y)\) satisfies:

  1. \(d(x,y) = 0 \iff x = y\)
  2. \(d(x,y) = d(y,x)\)   (Symmetry)
  3. \(d(x,z) \leq d(x,y) + d(y,z)\)   (Triangle Inequality)

Any set following these rules is a Metric Space.

Symmetry

When symmetries act, new shapes emerge

What is a Group?

A Group is a mathematical object that captures the idea of "composing symmetries."

  1. Composition: You can combine two symmetries
  2. Identity: "Doing nothing" is a symmetry
  3. Inverse: Every symmetry can be undone

The Circle Quotient

Glue together points that differ by an integer — we wrap the line into a Circle.

The hidden geometry of the Prime numbers

How Big is a Million?

In the \(p\)-adic numbers, size is determined by divisibility by a prime \(p\).

Real Size

1,000,000 is huge.

2-adic Size

\(10^6 = 2^6 \cdot 5^6\)
It is divisible by 2 many times.
It is tiny.

This completely inverts our intuition of "far" and "near."

The \(p\)-adic Tree

The \(p\)-adic numbers have a fractal, tree-like geometry. Every rational rotation "shadows" onto one of these trees.
