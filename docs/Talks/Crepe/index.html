Metric Spaces

What is space?

Mathematics begins with abstraction. We take a familiar idea — distance — and strip it to its absolute essentials.

A set of points where a distance \(d(x,y)\) satisfies:

  1. \(d(x,y) = 0 \iff x = y\)
  2. \(d(x,y) = d(y,x)\)   (Symmetry)
  3. \(d(x,z) \leq d(x,y) + d(y,z)\)   (Triangle Inequality)

Any set following these rules is a Metric Space.

Symmetries

What is a symmetry?

Suppose we have a geometric space (call it \(X\)). A symmetry is just some rigid motion of that space back onto itself that preserves all of the distances between points.

  1. Composition: You can follow one symmetry by another
  2. Identity: "Doing nothing" is a symmetry
  3. Inverse: Every symmetry can be undone

If we start with a few symmetries, we can consider all compositions of them and their inverses.

Once we have collected all such symmetries, we call this a group of symmetries.

Quotients

What is a quotient?

Suppose we have a geometric space (call it \(X\)), and some initial symmetries.

From this input, we can build the group of symmetries \(G\) by collecting all symmetries built out of our original ones.

We think of the group of symmetries as dividing our space up into a bunch of identical regions. Since we are dividing the space up, we can call one of these regions the quotient \(X/G\).
