Cayley graph from explicit elements of SO₃(ℤ[∛2])

This version uses an actual SL₂(ℂ) action on hyperbolic space. A word g acts on the basepoint via X ↦ g X g* in the Hermitian model of H³, then the resulting point is sent to the Poincaré ball.

Here ∛2 is embedded into ℂ by α ↦ 2^(1/3) e^{2π i / 3}, and we use the nontrivial identity (2-α)² + (1+α-α²)² = 1.

g₁ = q(x,y,0) g₂ = q(x,0,y) g₃ = q(0,x,y)

with x = 2-α, y = 1+α-α², and q(b,c,d) the pure unit quaternion bi+cj+dk, mapped to SL₂(ℂ).
