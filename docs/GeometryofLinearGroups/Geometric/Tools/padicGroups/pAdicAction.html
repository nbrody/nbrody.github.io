Matrix Action on $T_p$

This tool analyzes the action of $\mathsf{PGL}_2(\mathbb{Q})$ on the Bruhat-Tits tree $T_p$. A vertex is denoted by $\lfloor q \rfloor_k$, where $k\in \mathbb{Z}$ and $q\in \mathbb{Z}[1/p]/p^k\mathbb{Z}$.
