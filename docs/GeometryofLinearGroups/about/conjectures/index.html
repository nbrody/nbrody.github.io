Geometry of Linear Groups

Conjecture 1

Let \(k\) be a number field and suppose that \(\mathbb{G}\) is a simple \(k\)-algebraic group. We call a subgroup \(\Gamma\leq \mathbb{G}(k)\)

(1) Algebraic if there is a proper algebraic subgroup \(\mathbb{H} \lneq \mathsf{Res}|_{k/\mathbb{Q}} \mathbb{G}\) with \(\Gamma \leq \mathbb{H}(\mathbb{Q})\)
(2) Geometric if there is a valuation \(v\) on \(k\) so that \(\Gamma\) is discrete in \(\mathbb{G}(k_v)\)
(3) Arithmetic if there is a subring \(A\leq k\) so that \(\Gamma\) is commensurable with \(\mathbb{G}(A)\).

We conjecture that every finitely generated subgroup of \(\mathbb{G}(k)\) is either algebraic, geometric, or arithmetic.

Conjecture 2

Suppose that \(\Gamma\leq \mathbb{G}(k)\) is a geometric subgroup. Then \(\Gamma\) has a finite-index subgroup which is built from arithmetic pieces.
