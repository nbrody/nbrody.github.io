Generators for Projective Units

About

This tool computes generators for the group of projective units of a quaternion algebra defined by a Hilbert symbol (a,b)_A over a number ring A.

The quaternion algebra is the ℝ-algebra generated by elements i, j with relations:

  • i² = a
  • j² = b
  • ij = -ji
