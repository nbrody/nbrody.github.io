# Bay Area — Research References

This folder holds the topographic and layout research behind the Bay Area
regional terrain (`bayArea.js`).

**Status:** stub. The regional terrain was built before the `refs/`
convention was introduced. If you work on the Bay Area terrain, please
backfill the research here.

UC Berkeley, which sits inside this region, has its own refs folder at
[`../berkeley/refs/`](../../berkeley/refs/).

See [`../../LEVELS.md`](../../LEVELS.md) for the refs folder convention.

## What to document here

- **SF Bay shape and bathymetry** — coastline, shallow South Bay vs. deeper
  Central Bay, wetland fringes.
- **Topography** — Diablo Range (east), Santa Cruz Mountains (south), Marin
  Headlands (north), Berkeley/Oakland Hills.
- **Bridges** — Bay Bridge, Golden Gate, San Mateo, Richmond–San Rafael,
  Dumbarton.
- **City locations** — San Francisco, Oakland, Berkeley, Palo Alto, San Jose,
  Marin County cities.
