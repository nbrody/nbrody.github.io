Surface Groups in PGL₂(ℤ[1/p])

Construct explicit surface groups by searching for a transitive permutation representation of F₂ = ⟨a,b⟩ of degree 2p+2 whose commutator u=[a,b] has cycle type (2)(2p). The resulting subgroup H < ⟨A,B⟩ has two cusps, which are glued by τ = diag(p,1) to produce a closed surface group.

⚙ Parameters
𝐌 Base Matrices
A = [[2, 1], [1, 1]] B = [[2, -1], [-1, 1]] U = [A,B] = [[1, 6], [0, 1]] τ = [[p, 0], [0, 1]]

The subgroup H is computed via a Schreier transversal. Its generators are given as words in A, B and as explicit integer matrices.
