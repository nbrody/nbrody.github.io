About the method

Each step uses the leapfrog stencil un+1 = 2un − un−1 + κ²·∇²un, where κ = c·dt/h is the dimensionless wave speed; CFL stability in 2D requires κ ≤ 1/√2. Walls clamp u to zero (perfect reflectors). Absorbing boundaries use a Mur 1st-order condition that lets outgoing waves leave the domain without reflecting.
