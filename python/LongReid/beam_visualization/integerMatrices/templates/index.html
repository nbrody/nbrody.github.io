Integer Matrix Search

Find words evaluating to integer matrices in surface subgroups of \(PSL(2,\mathbb{Q})\)

About This Tool

This tool explores surface subgroups of \(PGL(2, \mathbb{Q})\). We consider matrices generated by \(a\) and \(b\) which depend on a parameter \(t\).

\[ a = \begin{pmatrix} t & 0 \\ 0 & 1 \end{pmatrix}, \quad b = \begin{pmatrix} 1+t^2 & 2 \\ t & 1 \end{pmatrix} \]

Viewing \(t\) as an indeterminate, these matrices generate a subgroup of \(PGL(2, \mathbb{Z}[t,\frac{1}{t},\frac{1}{t-1}])\). The commutator has trace zero, and so has order 2. There is a specialization of the group which determines a discrete and faithful representation into \(PSL(2, \mathbb{R})\), and we conclude that this is a surface subgroup of \(PGL(2, \mathbb{Z}[t,\frac{1}{t},\frac{1}{t-1}])\).

The goal is to find non-trivial group elements that result in integer matrices with determinant \(\pm 1\). The search uses a beam search algorithm to efficiently explore the group and identify these "integral points" for a given value of \(t\).
